Welcome to Caliburn.Micro.Windsor

Caliburn.Micro.Windsor is a nuget package to easily get started with Caliburn.Micro and Castle Windsor.

More about caliburn micro can be found at: http://caliburnmicro.codeplex.com

More about Castle Windsor can be found at: http://www.castleproject.org

Any issues with the package can be reported at https://github.com/marwijn/Caliburn.Micro.Windsor/issues
